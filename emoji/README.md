# Engenharia de Soluções Lógicas 🧠💻

Este repositório contém uma série de estudos e scripts desenvolvidos para resolver problemas de automação, logística e processamento de dados, demonstrando o ciclo completo desde a modelagem lógica até o código final.

## 📁 Estrutura do Repositório

* **`root/`**: Arquivo de apresentação principal.
* **`/solucoes-logicas/`**: Pasta contendo o código-fonte, documentação técnica e estudos de caso.

---

## 🚀 Projetos e Módulos

### 1. Logística e Otimização de Entregas 🚚
[cite_start]Desenvolvimento de uma lógica para gestão de frotas e estafetas, baseada em documentos técnicos e fluxogramas[cite: 1, 27].
* [cite_start]**Cálculo de Tempo**: Algoritmo que projeta o tempo de entrega com base na distância e velocidade média, aplicando um ajuste de 1.5x em situações de trânsito alto[cite: 36, 37].
* [cite_start]**Gestão de Prioridades**: Sistema de pesos para garantir que entregas prioritárias ocupem o topo da lista[cite: 40, 41].
* [cite_start]**Cenários de Teste (QA)**: Validação de casos ideais, urgentes e tratamento de erros para distâncias inválidas (negativas)[cite: 57, 62, 68].

### 2. Processamento Digital de Imagem (Baixo Nível) 🎨
* **Renderizador de Emoji**: Script que manipula uma grade de pixels $5 \times 5$ (matriz RGB).
* **Filtros Matemáticos**: Aplicação de um algoritmo para reduzir a intensidade dos canais de cor em 50%, demonstrando como aplicar filtros de brilho via manipulação direta de dados.

### 3. Automação e Validação de Dados 📊
* **Limite de Segurança**: Ferramenta para análise de média de vendas que valida se os resultados estão dentro de parâmetros operacionais seguros.
* [cite_start]**Tratamento de Exceções**: Implementação de lógica para impedir a entrada de valores negativos ou tipos de dados incorretos no sistema[cite: 81, 82].

## 🛠️ Tecnologias Utilizadas

* **Linguagem**: Python 3.x.
* **Bibliotecas**: `matplotlib` para renderização gráfica e visualização de matrizes.
* [cite_start]**Documentação Lógica**: Fluxogramas operacionais e Pseudocódigo[cite: 1, 27].

## ✒️ Metodologia
[cite_start]O projeto foca na **Engenharia de Soluções**, onde cada problema é primeiramente desenhado logicamente (Fluxograma) [cite: 1][cite_start], estruturado em linguagem humana (Pseudocódigo) [cite: 27] e, por fim, codificado para execução em ambientes como o Google Colab.
